# 魔塔生日小游戏 · 关卡设计稿（完整版）

> **核心彩蛋**：L2 有「神秘宝物★」被黄门锁住，L1 唯一的黄钥匙。  
> 贪心拿★ → L4 黄门打不开 → 资源不足 → BOSS 必败。  
> 不拿★ → L4 黄门资源全拿 → BOSS 惊险通关。

### 升级规则
- 经验满 **80** 自动升级 → HP+30，攻+4，防+3，经验清零

### 战斗公式
- 玩家伤害 = max(1, 玩攻 − 怪防)
- 怪物伤害 = max(0, 怪攻 − 玩防)
- 击杀回合 = ceil(怪HP / 玩伤)，玩家先攻
- 玩家承伤 = (击杀回合 − 1) × 怪伤
- HP ≤ 承伤 → 失败

## 设计核心总结

**为什么拿★反而输？**

1. ★ 的实际收益：+60HP +4攻 +2防（看起来很强）
2. 但代价是用掉黄钥匙，永久失去 L4 黄门后的：+70HP(蓝药) +3攻(攻宝石)
3. 同时错过 L2 安全路线的：+70HP(蓝药) +3防(防宝石)
4. 净损失：**-80HP -2攻 -1防**
5. 面对 BOSS 时恰好差那几十点 HP → 必败

**反直觉的精髓**：★ 给的数值「看起来」很大（+60HP！+4攻！），但它的**机会成本**（丧失两处关键资源）更大。小朋友第一次玩会本能拿★，失败后复盘才能理解「贪心的代价」。
