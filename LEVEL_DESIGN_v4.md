### 升级
- 经验满 **100** → HP+100, 攻+5, 防+5

### 战斗
- 玩家伤害 = max(1, 玩攻 − 怪防)
- 怪物伤害 = max(0, 怪攻 − 玩防)
- 杀怪 ceil(怪HP / 玩伤) 次，玩家先攻
- 玩家承伤 = (次数 − 1) × 怪伤

## L1 教学层（13×13）

**设计意图**：教玩家基础操作——走路、捡钥匙、开门、打怪。窄走廊结构。
末尾给 1 把红钥匙（让小朋友"舍不得不开"到 L2 用）。

```
#############

#.###.###.#.#
#.#.......#.#
#.#.#####.#.#
#.Z.#...#.Z.#
#.#.#.#.#.#.#
#.#.h.#...#.#
#.#.#####.#.#
#.#.y.....#.#
#.#.#####Y#.#
#.r.......U.#
#############
```

**物件清单**：
- 僵尸 Z ×4: (4,1) (10,1) (2,5) (10,5)
- 黄钥匙 y ×2: (7,1) (4,9)
- 红药水 h: (4,7)
- 黄门 Y: (10,10)
- 红钥匙 r: (2,11) — **L1 唯一红钥匙，带到 L2 用**
- 上楼梯 U: (11,11)

**钥匙统计**：黄 2 把，黄门 1 道 → 余 1 把带走。红 1 把，本层无红门。
**验证要求**：
- 不开门可到达：所有 Z、y、h、r
- 开 1 黄门后可到达：U
- 主路：@ → Z → y → Z → h → y → Z → Z → 开Y → r → U

## ★ 陷阱机制

- L1 给 1 把红钥匙 r
- L2 有 1 道红门 R，后面是 ★ 神秘宝物
- 贪心开 → L2 拿★ → BOSS 前属性低 → 必败
- 聪明不开 → BOSS 前回 L2 拿★ → 全属性翻倍 → 险胜
- 不拿★ → 攻 < BOSS 防 → 必败
